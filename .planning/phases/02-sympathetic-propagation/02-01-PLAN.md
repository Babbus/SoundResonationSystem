**System struct:**
- `[BurstCompile]`
- `[UpdateInGroup(typeof(SimulationSystemGroup))]`
- `[UpdateAfter(typeof(EmitterActivationSystem))]`
- `[UpdateBefore(typeof(ExponentialDecaySystem))]`
- `public partial struct SympatheticPropagationSystem : ISystem`

**Private field:**
- `private EntityQuery _emitterQuery;`

**OnCreate:**
- Build `_emitterQuery` using `new EntityQueryBuilder(Allocator.Temp).WithAll<EmitterTag, ResonantObjectData, LocalTransform>().Build(ref state);`
- Note: This query uses DEFAULT filtering (only entities with ENABLED EmitterTag), which is correct -- we only want active emitters in the snapshot.

**OnUpdate (Pass 1 -- main thread emitter collection):**
1. Get emitter count: `int emitterCount = _emitterQuery.CalculateEntityCount();`
2. Early exit if `emitterCount == 0` (no active emitters, nothing to propagate)
3. Collect emitter data using `_emitterQuery.ToComponentDataArray<ResonantObjectData>(Allocator.TempJob)` and `_emitterQuery.ToComponentDataArray<LocalTransform>(Allocator.TempJob)`
4. Build `NativeArray<EmitterSnapshot>(emitterCount, Allocator.TempJob)` by copying Position, NaturalFrequency, CurrentAmplitude from the arrays
5. Get `float dt = SystemAPI.Time.DeltaTime;`

**OnUpdate (Pass 2 -- schedule parallel receiver job):**
6. Create and schedule `new PropagationJob { Emitters = snapshots, EmitterCount = emitterCount, DeltaTime = dt }.ScheduleParallel();`
7. Dispose all NativeArrays using `.Dispose(state.Dependency)` to chain disposal after job completion. Dispose: emitterData, emitterTransforms, and snapshots.

**PropagationJob (nested partial struct):**
- `[BurstCompile]`
- `[WithOptions(EntityQueryOptions.IgnoreComponentEnabledState)]` -- MUST iterate ALL entities including disabled EmitterTag
- `private partial struct PropagationJob : IJobEntity`
- Fields: `[ReadOnly] public NativeArray<EmitterSnapshot> Emitters;`, `public int EmitterCount;`, `public float DeltaTime;`
- Visibility floor constant: `private const float VisibilityFloor = 0.001f;` -- thesis demo convenience, not a physics parameter

**Execute method signature:**
```csharp
private void Execute(ref ResonantObjectData data, in LocalTransform transform, EnabledRefRW<EmitterTag> emitterEnabled)
```
Note: Use `ref ResonantObjectData` (not `in`) because we write CurrentAmplitude. Use `in LocalTransform` (read-only). Do NOT add `in EmitterTag` -- would alias with `EnabledRefRW<EmitterTag>` (Phase 1 lesson).

**Execute method body:**
1. Early exit: `if (EmitterCount == 0) return;`
2. Skip active emitters (direct-only, no cascade): `if (emitterEnabled.ValueRO) return;`
3. Initialize `float totalDrivingForce = 0f;` and `float3 receiverPos = transform.Position;`
4. Loop over all emitters (`for (int i = 0; i < EmitterCount; i++)`):
   a. Frequency culling: `float freqRatio = emitter.NaturalFrequency / data.NaturalFrequency;` -- skip if `freqRatio < 0.5f || freqRatio > 2.0f`
   b. Distance: `float dist = math.distance(receiverPos, emitter.Position);`
   c. Distance culling: `if (dist * dist > 100f) continue;` -- skip beyond 10m (squared to avoid sqrt in culling)
   d. Lorentzian response: `float response = ResonanceMath.LorentzianResponse(emitter.NaturalFrequency, data.NaturalFrequency, data.QFactor);`
   e. Inverse-square attenuation: `float attenuation = ResonanceMath.InverseSquareAttenuation(dist);`
   f. Accumulate: `totalDrivingForce += emitter.CurrentAmplitude * response * attenuation;`
5. If `totalDrivingForce <= 0f` return (no emitter contributed)
6. Apply visibility floor AFTER the > 0 check: `totalDrivingForce = math.max(totalDrivingForce, VisibilityFloor);`
7. Driven oscillator step: `float newAmplitude = ResonanceMath.DrivenOscillatorStep(data.CurrentAmplitude, totalDrivingForce, DeltaTime, data.NaturalFrequency, data.QFactor);`
8. Write: `data.CurrentAmplitude = newAmplitude;`
9. Enable EmitterTag if above threshold: `if (data.CurrentAmplitude > data.DeactivationThreshold) emitterEnabled.ValueRW = true;`
