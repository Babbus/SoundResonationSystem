    **VoiceSynthesizer.cs** (in `Assets/SoundResonance/Runtime/Audio/`):
    Create a MonoBehaviour in namespace `SoundResonance` that goes on each pooled AudioSource GameObject:

    **Fields:**
    - `private ResonanceAudioBridge _bridge;` -- set by bridge during pool init
    - `private int _voiceIndex;` -- which slot in the double-buffer this voice reads (set by bridge)
    - `private float[] _partialPhases = new float[HarmonicProfile.PartialCount];` -- phase accumulators per partial
    - `private float _envelopeGain = 0f;` -- current envelope level [0,1] for attack/release
    - `private bool _wasActive = false;` -- track activation state for attack/release transitions
    - `private float _transientTimer = 0f;` -- remaining transient duration in seconds
    - `private float _transientDuration = 0f;` -- total transient duration for current strike
    - `private float _transientPhase = 0f;` -- phase accumulator for transient bandpass
    - `private float _transientAmplitude = 0f;` -- strike amplitude for transient intensity
    - `private System.Random _rng;` -- noise generator (seeded with voiceIndex for determinism)
    - Constants: `private const float AttackTime = 0.002f;` (2ms), `private const float ReleaseTime = 0.075f;` (75ms), `private const float TransientMix = 0.3f;`

    **Init method:**
    - `public void Initialize(ResonanceAudioBridge bridge, int voiceIndex)`: store references, seed _rng with voiceIndex

    **OnAudioFilterRead(float[] data, int channels):** (called on audio thread)
    1. Read voice data from bridge: `var readBuffer = _bridge.GetReadBuffer(); var voiceData = readBuffer[_voiceIndex];`
    2. Get sample rate: `int sampleRate = AudioSettings.outputSampleRate;`
    3. Compute `float increment = 2f * Mathf.PI / sampleRate;`
    4. Get harmonic ratios and weights:
       ```csharp
       float[] ratios = HarmonicProfile.GetRatios(voiceData.Shape);
       float[] weights = HarmonicProfile.GetWeights(voiceData.Shape, voiceData.IsSympathetic == 1);
       ```

    5. Handle activation/deactivation state transitions:
       - If `voiceData.Active == 1` and `!_wasActive`: voice just activated. If `voiceData.IsNewStrike == 1`, trigger transient (set _transientTimer, _transientDuration based on shape: Bar=0.005f, Shell=0.003f, Plate=0.012f, _transientAmplitude = voiceData.StrikeAmplitude, _transientPhase = 0). Set _wasActive = true.
       - If `voiceData.Active == 0` and `_wasActive` and `_envelopeGain <= 0.0001f`: release complete, set _wasActive = false, zero the buffer and return.
       - If `voiceData.IsNewStrike == 1` and `_wasActive`: re-strike on already active voice. Trigger transient but do NOT reset partial phases (prevents click). Update _transientAmplitude.

    6. Per-sample loop: `for (int i = 0; i < data.Length; i += channels)`
       a. **Envelope update:**
          - If `voiceData.Active == 1`: ramp _envelopeGain toward 1.0 with attack time constant: `_envelopeGain += (1f - _envelopeGain) * Mathf.Min(1f, 1f / (AttackTime * sampleRate));`
          - Else: ramp _envelopeGain toward 0.0 with release time: `_envelopeGain *= 1f - Mathf.Min(1f, 1f / (ReleaseTime * sampleRate));`
          - If _envelopeGain < 0.00001f during release, snap to 0.

       b. **Additive sine synthesis:**
          ```csharp
          float sample = 0f;
          for (int p = 0; p < HarmonicProfile.PartialCount; p++)
          {
              float freq = voiceData.Frequency * ratios[p];
              // Per-partial faster decay: partial p+1 decays (p+1)x faster
              // Use amplitude^(1 + p * 0.5f) to approximate faster decay for upper partials
              // When amplitude is 0.5: partial 0 = 0.5, partial 1 = 0.35, partial 2 = 0.25, partial 3 = 0.18
              float partialDecay = Mathf.Pow(voiceData.Amplitude, 1f + p * 0.5f);
              float partialAmp = partialDecay * weights[p];
              sample += Mathf.Sin(_partialPhases[p]) * partialAmp;
              _partialPhases[p] += freq * increment;
              if (_partialPhases[p] > 2f * Mathf.PI)
                  _partialPhases[p] -= 2f * Mathf.PI;
          }
          ```

       c. **Strike transient:**
          ```csharp
          if (_transientTimer > 0f)
          {
              float noise = (float)(_rng.NextDouble() * 2.0 - 1.0);
              float bandpass = noise * Mathf.Sin(_transientPhase);
              _transientPhase += voiceData.Frequency * increment;
              if (_transientPhase > 2f * Mathf.PI)
                  _transientPhase -= 2f * Mathf.PI;
              float envelope = _transientTimer / _transientDuration;
              sample += bandpass * envelope * _transientAmplitude * TransientMix;
              _transientTimer -= 1f / sampleRate;
          }
          ```

       d. **Apply envelope and write to all channels:**
          ```csharp
          sample *= _envelopeGain;
          // Master volume scale to prevent clipping with multiple voices
          sample *= 0.15f;
          for (int ch = 0; ch < channels; ch++)
              data[i + ch] = sample;
          ```

    **Critical rules (from RESEARCH.md anti-patterns):**
    - NO allocations in OnAudioFilterRead (no `new`, no LINQ, no strings)
    - NO locks in OnAudioFilterRead
    - NEVER reset _partialPhases on frequency change or re-strike (prevents clicks)
    - Always write to ALL channels in the interleaved buffer
    - Phase wrapping uses subtraction (`-= 2*PI`), not modulo

    **Wire into ResonanceAudioBridge:**
    Add to ResonanceAudioBridge.cs:
    - `private VoiceSynthesizer[] _synthesizers;` -- array of 16
    - In `Awake()`, after allocating buffers and creating VoicePool, instantiate 16 child GameObjects:
      ```csharp
      _synthesizers = new VoiceSynthesizer[MaxVoices];
      for (int i = 0; i < MaxVoices; i++)
      {
          var voiceGO = new GameObject($"Voice_{i}");
          voiceGO.transform.SetParent(transform);
          var audioSource = voiceGO.AddComponent<AudioSource>();
          audioSource.spatialBlend = 1f; // full 3D
          audioSource.rolloffMode = AudioRolloffMode.Logarithmic;
          audioSource.minDistance = 1f;
          audioSource.maxDistance = 50f;
          audioSource.dopplerLevel = 0f; // disable doppler for synthesis
          audioSource.playOnAwake = false;
          audioSource.loop = true; // keep callback firing
          audioSource.clip = null; // no clip, we generate PCM
          var synth = voiceGO.AddComponent<VoiceSynthesizer>();
          synth.Initialize(this, i);
          _synthesizers[i] = synth;
          audioSource.Play(); // MUST call Play() for OnAudioFilterRead to fire
      }
      ```
    - In `OnDestroy()`, destroy child voice GameObjects (they are children, so DestroyImmediate/Destroy on this.gameObject handles it, but explicitly clean up if needed)
    - In `LateUpdate()`, after writing to the write buffer, sync AudioSource positions:
      ```csharp
      var readBuf = GetReadBuffer(); // the buffer that was JUST swapped to be the new read buffer
      // Actually, after swap the NEW read buffer is the one we just wrote to.
      // Use the writeBuffer reference (before swap) since that's what we just populated.
      for (int i = 0; i < MaxVoices; i++)
      {
          if (writeBuffer[i].Active == 1)
              _synthesizers[i].transform.position = (Vector3)writeBuffer[i].Position;
      }
      ```
      This position sync must happen BEFORE the _readIndex swap, using the writeBuffer data.
